What's the difference between Web 1.0, Web 2.0 and Web 3.0?
Web 1.0 is just an information portal, passive interaction where users gets information without being able to post reviews, comments, and feedback.

Web 2.0 wants the user to create interaction. Meaning- participation, collaboration, and information sharing. Examples of Web 2.0 applications are Youtube, Wiki, Flickr, Facebook, and so on.

Web 3.0 is a semantic web which acts as smart humans. It can proccess information like humans, gets insignts regarding the users needs, and generate useful content which suits to the needs of users. One example of Web 3.0 is Tivo, a digital video recorder. Its recording program can search the web and read what it finds to you based on your preferences.

Use case Example demonstrates the differences: While searching for Side Effects of Medication, in web 1.0, the answer will be plain text, with no abillity to ask or interact. In contrast, in Web 3.0 its can search the web and read what it finds to you based on your preferences.
